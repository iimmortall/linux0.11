# linux0.11
linux0.11 source code. Including X86 simulator Bochs. 

hdc-0.11.img内包含有：

Bash shell
一些基本的Linux命令、工具，比如cp、rm、mv、tar。
vi编辑器
gcc 1.4编译器，可用来编译标准C程序
as86和ld86
Linux 0.11的源代码，可在0.11下编译，然后覆盖现有的二进制内核
